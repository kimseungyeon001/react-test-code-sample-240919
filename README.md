## Getting Started

### Install

create the project

```
git clone https://github.com/kimseungyeon001/react-test-code-sample-240919.git
```

access the project directory

```
cd react-test-code-sample-240919
```

install dependencies

```
npm install
```

setting config.ts

```
cp ./src/config.local.ts ./src/config.ts
```

### Start

```
npm run dev
```

### Storybook

```
npm run storybook
```

### Test

```
npm run test
or
npm run test:watch
```
